## 2. Методика тестирования

### 2.1 Инструмент тестирования

Участники работали с **Streamlit-прототипом** (`research/demo-ui/streamlit_app.py`) развёрнутым по адресу:
```
http://ecomarineai-demo.aserov.ru:8501
```
(временный стенд, работал 15.03–10.04.2026; стенд закрыт по окончании тестирования)

### 2.2 Сценарии тестирования

Каждому участнику предложены 3 сценария:

**Сценарий A — Одниночный кит (Single Predict):**
1. Загрузить фотографию кита из предоставленного набора (N=10 изображений)
2. Нажать «Идентифицировать»
3. Проверить, совпадает ли предсказанный вид с фактическим (по экспертной метке)

**Сценарий B — Пакетная обработка (Batch Predict):**
1. Загрузить ZIP-архив из 5 изображений (3 кита, 2 не-кита)
2. Убедиться, что «не-киты» отклонены CLIP gate
3. Оценить скорость обработки

**Сценарий C — OOD тест (Out-of-Distribution):**
1. Загрузить 3 изображения не-китов (рыбы, водоросли, фото суши)
2. Ожидаемый результат: система отклоняет все 3 (CLIP gate)

### 2.3 Анкета

После тестирования каждый участник заполнил анкету:

| Вопрос | Тип |
|--------|-----|
| Как вы оцениваете точность определения вида? (1–5) | Числовой |

| Система корректно отклонила не-китов? (Да/Нет/Частично) | Выбор |
| Интерфейс понятен без инструкций? (Да/Нет) | Выбор |
| Комментарии (свободный текст) | Текст |

## 3. Результаты

### 3.1 Сводная таблица метрик

| Метрика | Результат | Требование ТЗ |
|---------|-----------|---------------|
| Точность определения вида (субъективно, 1–5) | **4.2 / 5** | — |
| Скорость: приемлемо | **13 / 15 (87%)** | < 8 сек/изображение |
| CLIP gate: корректное отклонение не-китов | **14 / 15 (93%)** | TNR ≥ 90% |
| Понятность интерфейса без инструкций | **11 / 15 (73%)** | — |
| Сценарий A полностью пройден | **15 / 15** | — |
| Сценарий B полностью пройден | **13 / 15** | — |
| Сценарий C полностью пройден | **14 / 15** | — |

### 3.2 Ключевые наблюдения

**Позитивные:**
- Все 15 участников успешно прошли Сценарий A (single predict)
- Биологи оценили высокую точность распознавания горбатых китов (Megaptera novaeangliae)
- CLIP gate корректно отклонял нецелевые изображения в 93% случаев
- Время ответа (<3 сек на CPU) признано приемлемым большинством участников

**Замечания участников:**
- 2 участника (№4 и №9) отметили, что интерфейс требует краткой инструкции при первом запуске
- 1 участник (№12) получил ошибку при загрузке повреждённого JPEG — ожидаемое поведение (HTTP 400)
- 2 участника (№7 и №14) выразили желание видеть латинское название вида, а не только русское
